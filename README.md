# Quiz App

A simple Quiz App built using HTML, CSS, and JavaScript.

## Features

- Multiple choice questions
- Score calculation
- Correct and wrong answer highlight
- Progress bar
- Restart quiz option
- Responsive design

## Tech Stack

- HTML
- CSS
- JavaScript

## How to Run

1. Clone the repository
2. Open `index.html` in browser
3. Start the quiz
